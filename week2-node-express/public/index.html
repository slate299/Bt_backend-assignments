POST /user

Creates a user profile with JSON data

Required JSON:

{ "id": "123", "name": "John Doe" }

Usage:

curl -X POST http://localhost:3000/user \ -H "Content-Type: application/json" \ -d '{"id":"123","name":"John Doe"}'

Note: 'id' and 'name' are required fields. Sending either missing will return a 400 error.
